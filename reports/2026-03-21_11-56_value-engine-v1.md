# REPORT: value-engine-v1

## 1. Summary
Реализован `ValueEngine v1` для расчета expected value по исходам `1/X/2`, выбора `best_bet` с максимальным EV и выставления `value_flag` по порогу.

## 2. Goal
Добавить простой и проверяемый слой value-оценки, который из вероятностей и коэффициентов получает количественную оценку преимущества ставки.

## 3. Scope
В рамках этапа:
- добавлен модуль `core/value/value_engine.py`;
- добавлены тесты `tests/test_value_engine.py`;
- добавлена валидация входных `probs`/`odds`.

Вне scope:
- изменение decision-логики;
- изменение модели вероятностей;
- изменение API-контрактов вне value-слоя.

## 4. Files Changed
- `core/value/value_engine.py`
- `tests/test_value_engine.py`
- `reports/2026-03-21_11-56_value-engine-v1.md`

## 5. Detailed Changes
1. Реализованы формулы:
   - `EV1 = P1 * O1 - 1`
   - `EVX = PX * OX - 1`
   - `EV2 = P2 * O2 - 1`
2. Добавлен выбор `best_bet` как исхода с максимальным EV.
3. Добавлен `value_flag=True`, если `max(EV) > 0.05`.
4. Добавлены проверки валидности входов:
   - обязательные ключи;
   - конечные значения без `NaN/Inf`;
   - диапазон вероятностей `[0, 1]` и сумма `1.0` с `abs_tol=1e-6`;
   - коэффициенты строго больше `1`.
5. Добавлено INFO-логирование входов и результата расчета.

## 6. Public Contracts / Interfaces
Контракт `ValueEngine.calculate(probs, odds)` возвращает:
- `EV: dict[str, float]`
- `best_bet: str`
- `value_flag: bool`

Для невалидных входов выбрасываются предсказуемые исключения валидации.

## 11. Manual Verification
Проверено вручную:

## 12. Risks / Known Issues
- Порог `0.05` может давать избыточные/недостаточные сигналы при иной марже букмекера.
- Чувствительность к калибровке вероятностей: при систематическом bias value-сигнал искажается.
- В текущей версии отсутствует поправка на ликвидность/лимиты рынка.

## 13. Deviations from Spec
Отклонений от целевой спецификации `ValueEngine v1` не зафиксировано.

Сильные стороны:

- строгое валидационное покрытие входов.

Ограничения:

План усиления:

- добавить калибровку по историческому ROI;
