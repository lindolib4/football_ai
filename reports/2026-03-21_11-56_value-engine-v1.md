# Value Engine v1 Report

## Что сделано
- Добавлен модуль `ValueEngine` в `core/value/value_engine.py`.
- Реализован расчет expected value по формулам:
  - `EV1 = P1 * O1 - 1`
  - `EVX = PX * OX - 1`
  - `EV2 = P2 * O2 - 1`
- Добавлена логика выбора `best_bet` как исхода с максимальным EV.
- Добавлена логика `value_flag = True`, если максимальный EV строго больше `0.05`.

## Валидации
- Проверка наличия обязательных ключей в `probs` и `odds`.
- Проверка `probs`:
  - конечные значения (без `NaN/Inf`),
  - диапазон `[0, 1]`,
  - сумма вероятностей равна `1.0` (с `abs_tol=1e-6`).
- Проверка `odds`:
  - конечные значения,
  - каждое значение строго больше `1`.

## Логирование
В `ValueEngine.calculate()` добавлено INFO-логирование:
- входных `probs`,
- входных `odds`,
- рассчитанных `EV`,
- итоговых `best_bet` и `value_flag`.

## Тесты
Добавлен файл `tests/test_value_engine.py` с проверками:
- корректности расчета EV,
- выбора `best_bet`,
- работы `value_flag`,
- валидации некорректных входов.

## Ограничения ТЗ
- ML не использовался.
- Модуль decision не изменялся.
- Реализация сохранена простой и прямой.
